修改readme.md文件，提交master主分支
1. 暂存所有文件 
    git add -A
2. 提交
    git commit -m "文件注释"
3. 推送
    git push
--------------------------------------------------------------------------------
本地添加dev分支(以下均为dev分支的操作，与master分支没有关系)
1. 选择git工作流,点击确定
2. 编辑文件
3. 暂存文件
    git add -A
4. 提交
    git commit -m "提交到dev分支"
5. 推送dev分支
    git push origin dev
